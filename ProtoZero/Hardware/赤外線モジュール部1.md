# Kagerou (陽炎)  
## 赤外線LEDモジュール回路 2026/4/15

### 赤外線LED駆動(発信)回路

```
             A   K
5V ---□□□□----|>|------+
       220Ω     LED1   |
5V ---□□□□----|>|------+
       220Ω     LED2   |
5V ---□□□□----|>|------+
       220Ω     LED3   |
                       | C
                     |/
RP2040 GPIO ---□□□□--|  2SC1815
               1kΩ  B|➘
                       | E
                       |
                      GND
```

| 図記号 | 補足 | 接続 |  
| ---- | ---- | ---- |
| 5V | VBUS | pin 23 |  
| GPIO | RP2040 GPIO2 | pin 3 |  
| IR LED1~3 | OSI5LA5113A | 長足:アノード(+)<br>短足:カソード(-) |  
| C-B-E | NPN: 2SC1815 | 平な面を向けて<br>左からE―C―B |  
| 220Ω | 1/4w 抵抗 | 150Ω程度までは強化OK |  
| 1kΩ | 1/4w 抵抗 |  |  

---

### 赤外線LED受信(受光)回路
```
                          Vcc +-----+
5V ------------□□□□-----------*     |
               100Ω      Vout | 赤外線受信モジュール
RP2040 GPIO -------|>|--------* PL-IRM2121-A538
             (A) 1N4148 (K)   |     |
                              +--+--+
  ※ VSとGND間にパスコン0.1μFを     |
     入れること。                  GND
```

| 図記号 | 補足 | 接続 |  
| ---- | ---- | ---- |
| 5V | VBUS | pin 23 |  
| GPIO | RP2040 GPIO3 | pin 4 |  
| 1N4148 | ダイオード | 線が無い方:アノード(+)<br>線が有る方:カソード(-) |  
| PL-IRM2121 | 赤外受光素子 | 受光面を向けて<br>左からVout- GND-Vcc |  
| パスコン | 0.1μF | 受光素子の近くに入れること |  
| 100Ω | 1/4w 抵抗 |  |
