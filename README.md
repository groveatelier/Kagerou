## 🌳 陽炎（Kagerou）について

陽炎は、  
音声認識はスマホ(Android)に実行させ、RP2040―Zero基板に  
付加した回路(赤外線リモコン)を通して家電を操作を実現させる  
**静かで存在感の薄いリモコンデバイス**です。

- スマホ(Android) : 専用(過去に使用していたものを活用)  
- エンジン： RP2040-Zero(学習リモコン本体)  
- 付加デバイス： 温度センサ，照度センサ，マイク etc...
- quietgrove の世界観に基づく、長く使える設計
